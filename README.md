# 🛍️ Products App

A responsive Products App built using **HTML, CSS, and JavaScript**.  
The application fetches product data from the **DummyJSON API** and provides search, 
category filtering, price sorting, and reset functionality.

## 🚀 Features

- 📦 Fetch products from DummyJSON API
- 🔍 Search products by title
- 🗂️ Filter products by category
- 💰 Sort products by price
  - Low to High
  - High to Low
- 🔄 Reset all filters
- 📱 Responsive and clean UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- Fetch API
- DummyJSON API
